# Walmart Sales Analysis (ML + ANN)

This project analyzes Walmart sales data and compares machine learning models with an ANN-based approach in a Jupyter notebook.

## Project Files

- `walmart.ipynb` — main notebook for cleaning, EDA, feature preparation, model training, and evaluation.
- `Walmart.csv` — original dataset.
- `Walmart_cleaned.csv` — cleaned/processed dataset used by the notebook.
- `ann-walmart-graph.png` — ANN result visualization.
- `model-comparison-walmart.png` — model performance comparison chart.
- `Walmart_Sales_Analysis_Report.docx` / `Walmart_Sales_Analysis_Report.pdf` — project report.

## Requirements

Use Python 3.9+ and install:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn tensorflow jupyter
```

## How to Run

1. Open the `walmart` folder in VS Code.
2. Open `walmart.ipynb`.
3. Select a Python kernel.
4. Run cells from top to bottom.

## Notes

- Keep dataset filenames unchanged (`Walmart.csv`, `Walmart_cleaned.csv`) so notebook paths work correctly.
- If TensorFlow is not available on your system, install it separately and restart the kernel.
